# Subtitle Hub

Subtitle Hub is a platform for managing and providing subtitles, consisting of an API, a user interface (UI), and a proxy to handle requests.

# Features

- API written in Golang

- UI developed with Next.js

- Proxy to provide access to both UI and API from a single address

# Installation & Setup

To set up the project using Docker, run the following commands:

# Build and prepare the containers
docker compose build

# Run the project
docker compose up

# Access

UI URL:https://localhost:1443/

API Documentation URL:https://localhost:1443/docs/index.html
